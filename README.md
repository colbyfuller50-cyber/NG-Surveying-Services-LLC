# N.G. Surveying Services LLC — Updated Modern Site

This package contains a modern, techy website for N.G. Surveying Services LLC, with a readable hero and Formspree-ready request form.

## How to use the logo
1. Replace `images/logo.png` with your transparent logo file. Keep the filename `logo.png`.
2. The logo is clickable and links back to the homepage.

## How to update Formspree
1. Create a form at https://formspree.io and get your endpoint URL (e.g., https://formspree.io/f/abcd1234)
2. Replace the placeholder action URL `https://formspree.io/f/your-form-id` in `request.html` and `contact.html` with your endpoint.

## Deploy to GitHub Pages
1. Create a GitHub repo and upload all files in this folder.
2. In repo Settings → Pages, choose branch `main` and root folder `/`.
3. Visit https://<your-username>.github.io/<repo-name>/ after a few minutes.
