# N.G. Surveying Services LLC Website

This is the official website for N.G. Surveying Services LLC, based in Columbia, South Carolina.

## Structure
- **index.html** — Home page with About, Services, and Gallery.
- **contact.html** — Contact page with form and Google Maps.
- **style.css** — Stylesheet for site layout and colors.
- **images/** — Contains logo and land surveying photos.

## GitHub Pages Instructions
1. Create a new GitHub repository (e.g., `ng-surveying-website`).
2. Upload all files from this ZIP into the repo.
3. Go to *Settings → Pages* and set the branch to `main` and folder to `/ (root)`.
4. Wait a few minutes, and your website will be live at:
   ```
   https://<your-username>.github.io/ng-surveying-website/
   ```
